# Setup
1. Create a storage account in Azure
2. Create a .env.local file containing:
```
NEXT_PUBLIC_STORAGE_ACCOUNT=<storage account name>
SECRET_ACCESS_KEY=<storage account key>
```

## Install dependencies
```
cd frontend
pnpm install
```
